Help

Welcome to the help section! You can view all of the commands available in this section!

 

 
Command format

All commands entered in TasKoord have a few keywords that have to be used. They are as follows:

 

1. Description - any words between the command word and the next key

2. Start time - after “from”

3. End time - after “to”

4. Deadline time - after “by”

5. Date - after “on”

6. Location - words after “at"

7. Priority - after "priority"

8. Status - after "status"

If you wish to include words such as "from" or "to" in your task description, simply add a / in front of the word!

An example is given as seen below:

add travel /from home /to school /on a bus /by PIE on 14 nov 2014 at Garden /by the bay

 


Adding timed tasks

You can add a timed task (a task with a start and end time) using commands 1-2.

You can also omit the date and the task will be added automatically to the current date using command 3.

You can also add a task that spans across a few days using command 4.

You can omit the time and the task will be assigned a time from 12am to 11.59pm of the date specified using command 5.

 

1. add timedTaskA from 3pm to 4pm on 15 nov 2014 at locationA

2. add timedTaskB from 0500h to 0600h on 15/11/14 at locationB

3. add timedTaskE from 1500h to 1600h at locationE

4. add timedTaskC from 3pm on 17 nov 2014 to 1600h on 18/11/14 at locationC

5. add timedTaskD on 18 Nov 2014 at locationD

 


Adding deadline tasks

You can add a task with a deadline using command 1.

You can omit the date and the task will be added to the current date using command 2.

 

1. add deadlineTaskA by 8pm on 16/11/14 at locationA

2. add deadlineTaskB by 8pm at locationB

 


Adding floating tasks

You can add good-to-do or floating tasks as shown below.

 

1. add floatingTaskA

 


Editing tasks

You can edit a task using the edit commands show below.

The standard format of an edit command is shown in 1.

You can edit a task to change it from a timed task to a deadline task and vice versa using commands 2-3.

The components of the task (i.e. description, start time, end time, deadline time, date, location, priority and status) can be individually edited as shown from commands 4-12.

 

1. edit timedTaskA correct from 4pm to 5pm on 15/11/14

2. edit timedTaskD correct deadlineTaskA by 3pm on 15/11/14

3. edit deadlineTaskB correct timedTaskE from 5pm to 6pm on 15/11/14

4. edit timedTaskB correct timedTaskC from 4pm to 5pm on 15/11/14

5. edit 1 correct timedTaskB

6. edit 2 correct from 3pm to 4pm

7. edit 3 correct by 5pm

8. edit 4 correct on 14/11/14

9. edit 5 correct from 3pm to 4pm on 14/11/14

10. edit 6 correct by 5pm on 14/11/14

11. edit 7 correct priority low (alternative to set)

12. edit 8 correct status completed (alternative to status)

 


Setting priority level and statuses

You can set the priority level of a task using commands 1-3. A task's priority can be set to high, medium and low and will be shown as red, teal and grey respectively.

You can also set the status of a task using commands 4-5. A task's status can be set to discarded, completed and pending and the status of the task will be displayed only when you list on a specific date or if you list a specific status, else the default status (pending) will not be shown when you list normally.

 

1. set 1 priority high

2. set 2 priority low

3. set 2 priority medium

4. set 1 status discarded

5. set 2 status completed

6. set 2 status pending

 

Clearing and deleting tasks

You can clear all your tasks in TasKoord using command 1.

You can also delete the tasks individually using keywords or the numbers that appear next to the task using commands 2-3.

 

1. clear

2. delete <number>

3. delete <task identification>

 

 

Undo and redo

You can undo the previous command you entered using command 1.

You can also redo the command you just undid using command 2.

***YOU CANNOT UNDO OR REDO THE 'SYNC' COMMAND***

 

1. undo

2. redo

 

 

Listing tasks

You can list all your tasks using command 1.

You can also list on a specific date or over a period of time using commands 2-3.

You can list tasks of a specific priority or task using commands 4-9.

You can list the different types of task (event, deadline and floating) using commands 10-12.

You can also use a combination of the different types of listing using commands 13-17.

 

1. list

2. list on 01/11/14

3. list from 1pm on 15 Nov 2014 to 1pm on 16 Nov 2014

4. list status completed

5. list status pending

6. list status discarded

7. list priority high (does not show discarded/completed task)

8. list priority medium (does not show discarded/completed task)

9. list priority low (does not show discarded/completed task)

10. list events

11. list deadlines

12. list floating

13. list <a specification> on <date>

14. list events and deadlines

15. list events and deadlines and floating

16. list upcoming deadlines priority high status completed

17. list upcoming <5 days/deadlines/events>

 


Searching for a task

You can search for a task using command 1.

 

1. search <keyword>

 


Sync

You can carry out synchronisation with Google Calendar using command 1.

 

1. sync

 


Exit

You can exit the program using command 1.

 

1. exit
